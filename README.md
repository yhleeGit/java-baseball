# 기능구현 진행 Step
## Step 01
* 컴퓨터가 입력받는 중복되지 않는 난수 생성

## Step 02
* 사용자가 입력하는 숫자 등록

## Step 03
* 사용자입력 숫자와 컴퓨터 입력 숫자에 대해 스트라이크 체크

## Step 04
* 사용자입력 숫자와 컴퓨터 입력 숫자에 대해 볼 체크

## Step 05
* 3스트라이크가 성립될때까지 게임 지속 코드 구현

## Step 06
* 게임이 종료 된 이후 다시 시작할 수 있는 코드 구현

## Step 07
* 각 입렵받는 값에 대한 벨리데이션 체크 및 사용성 개선
